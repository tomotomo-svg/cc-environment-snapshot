# cc-environment-snapshot

Claude Code 開発環境の棚卸しドキュメント置き場。

## このリポジトリについて

2026-06-29 時点の Claude Code 開発環境を、DIG部門メンバーへの共有・自身の整理・時点記録の3目的で文書化したもの。

## ファイル構成

| ファイル | 用途 |
|---|---|
| `current_development_environment.md` | Obsidian 取込み用 Markdown（wikilink / callout / TOC 付き） |
| `current_development_environment.html` | DIG部門メンバー配布用 HTML（スタンドアロン・オフライン閲覧可能） |

## 更新サイクル

環境に大きな変更があったタイミングで随時。ファイル名は常に同一（Git 履歴でバージョン管理）。

## 注意事項

- 社内プロジェクト名・顧客名等はマスク済み
- APIキー・認証情報は一切含まない
